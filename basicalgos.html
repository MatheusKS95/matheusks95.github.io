Basic Data Structures

This project was made with educational purposes. It shows how some basic data structures work (currently stack, queue and doubly-linked lists). It was written in C (C89 standard).

There are currently 3 projects (one for each data structure), with commentaries on every line to show what's going on. It also employs dynamic memory allocation and, for lists, two sorting algorithms (bubble sort and insertion sort) and one search algorithm (sequential).

Further information can be found here. This project will be discontinued soon, as I'm currently working in a new similar project (a C library this time), which can handle multiple data types (int, double, char, short, long, etc.).
