SpaceCrap 2

SpaceCrap 2 is a simple game that I wrote in C (C99 standard), using Raylib as a shared library. This is my first attempt at making a game in 3D without using any engine. Unlike the first game, which was based on an existing game, SpaceCrap 2 was entirely written from scratch (and trial-and-error, as I was exploring unknown territories with 3D).

This game is a sequel of the first SpaceCrap. It's a pretty simple story with space pirates and a defective spaceship that needs maintenance. In this game, the player needs to shoot down the pirates' leader. It's only a boss battle but gave me a good understanding of how 3D games work under the hood.

The game uses the keyboard as input (mouse is used in the main menu, however). Arrow keys move the ship, spacebar shoots and 1/2/3 (not the numpad ones) changes the player's weapon. You can leave at any time using Escape key. The gatling gun gets jammed if fired for too long. The game is in 3D but everything moves in a 2D plane (in order to keep it simple). SpaceCrap 2 also has two endings, based on who gets killed first.

For further information, including how to build, check it out here. As of today, SpaceCrap 2 runs on Linux only, and requires Raylib installed on your system as a shared library.
